2.3.1 Lenguajes de desarrollo web

HTML5, PHP, JavaScript y otros lenguajes esenciales.

Los lenguajes de desarrollo web son fundamentales para crear sitios web interactivos y dinámicos. Cada uno cumple un rol específico en la construcción de experiencias digitales.

LenguajeTipoDescripción
HTML5MarcadoEstructura el contenido de las páginas web con nuevas etiquetas para mejorar semántica y accesibilidad.
CSSEstilosDescribe la presentación de documentos HTML, controlando diseño y apariencia visual.
JavaScriptProgramación clienteLenguaje que se ejecuta en el navegador, permitiendo crear interactividad y manipular el DOM.
PHPProgramación servidorLenguaje de scripting del lado del servidor para desarrollar aplicaciones web dinámicas y gestionar bases de datos.
PythonProgramación servidorUtilizado con frameworks como Django o Flask para crear aplicaciones web dinámicas eficientes.
RubyProgramación servidorEspecialmente utilizado con Ruby on Rails para crear aplicaciones web rápidas y eficientes.
