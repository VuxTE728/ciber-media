1.3.3 El proceso en los diferentes tipos de animación

Cómo se lleva a cabo el proceso de animación.

El proceso de animación varía según el tipo, pero generalmente sigue una serie de etapas comunes.

Tipo de AnimaciónProceso de Animación
Animación 2D1. Conceptualización y guionización. 2. Diseño de personajes y fondos. 3. Creación de fotogramas clave. 4. Interpolación para fotogramas intermedios. 5. Edición y postproducción.
Animación 3D1. Modelado 3D de personajes y escenarios. 2. Rigging para esqueletos digitales. 3. Animación mediante keyframes o captura de movimiento. 4. Renderizado. 5. Edición y postproducción.
Stop Motion1. Creación de modelos físicos. 2. Configuración de escena e iluminación. 3. Fotografía cuadro por cuadro. 4. Edición para unir las imágenes en secuencia animada.
Animación por Computadora1. Diseño conceptual y guionización. 2. Creación de modelos digitales. 3. Animación con software especializado. 4. Renderizado. 5. Edición y postproducción con efectos visuales y sonido.
